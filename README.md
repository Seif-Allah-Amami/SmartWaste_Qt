# ♻️ Smart Application for Waste Collection and Recycling Management

C++ / Qt Project – 2A1

## 📌 Overview
This project is a **smart modular management system** for urban waste collection and recycling.  
It digitalizes waste operations, optimizes resources, and supports data-driven decision-making through analytics and reports.

The application is built using **C++ and Qt**, with a relational database backend.

---

## 🎯 Objectives
- Improve efficiency in waste collection and recycling
- Centralize all operations into one modular system
- Provide real-time statistics and reports
- Support sustainable development goals (SDGs)

---

## 🧩 Modules
The system is divided into independent but connected modules:

1. Employee Management  
2. Customer Reports Management  
3. Waste Management  
4. Vehicle Management  
5. Equipment (Machines) Management  
6. Supplier Management  

Each module includes:
- CRUD operations
- Search & sort
- Export to PDF/Excel
- Statistics & charts
- Role-based access
